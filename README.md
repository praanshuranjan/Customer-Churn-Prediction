# Customer Churn Prediction System (Milestone 1)

## Project Overview
[cite_start]This repository contains Milestone 1 of the Customer Churn Prediction & Agentic Retention Strategy project[cite: 1, 2, 3, 4]. [cite_start]The current implementation focuses on classical machine learning techniques to predict customer churn risk based on historical behavioral and transactional data[cite: 8, 42]. 

[cite_start]This milestone strictly utilizes traditional ML pipelines without the use of LLMs or Agentic AI[cite: 43, 97].

## Features
* [cite_start]**Data Preprocessing:** Handles missing values, scales numeric features, and encodes categorical variables using Scikit-Learn Pipelines[cite: 46, 49].
* [cite_start]**Machine Learning Models:** Trained using Logistic Regression and Decision Trees[cite: 49].
* [cite_start]**Interactive UI:** Built with Streamlit to accept customer profiles and instantly display churn probability and key drivers[cite: 32, 47].
* [cite_start]**Performance Evaluation:** Evaluated using Accuracy, Precision, Recall, F1 Score, and Confusion Matrix[cite: 50].

## Tech Stack
* **Language:** Python
* [cite_start]**UI Framework:** Streamlit [cite: 32]
* [cite_start]**Machine Learning:** Scikit-Learn [cite: 30]
